## Screenshot Tools Recommendations

### macOS
- **Built-in**: Shift + Command + 4 (area screenshot)
- **Recommended**: CleanShot X, Skitch

### Windows
- **Built-in**: Win + Shift + S (Snipping Tool)
- **Recommended**: Snagit, ShareX

### Linux
- **Built-in**: gnome-screenshot, flameshot
- **Recommended**: Flameshot, Shutter
