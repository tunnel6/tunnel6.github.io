## 账户管理

### 登录方式

YAT 支持以下登录方式：

- **GitHub** - 使用 GitHub 账号登录（推荐）
- **Google** - 使用 Google 账号登录
- **邮箱** - 使用邮箱注册（即将支持）

### 登录流程

1. 打开 YAT 客户端
2. 点击 **登录** 按钮
3. 选择登录方式
4. 授权访问
5. 完成登录

> 📸 **[截图位置]** 登录页面
> 
> 说明：显示 GitHub/Google 登录按钮

### 账户信息

登录后查看账户信息：

- **用户名**
- **邮箱**
- **头像**
- **注册时间**

### 退出登录

1. 点击 **Settings**
2. 点击 **退出登录**
3. 确认退出

> ⚠️ **注意**：退出登录后：
> - 隧道会停止
> - Edge 连接会断开
> - 需要重新登录才能使用

## 客户端设置

### 打开设置

点击左侧导航栏的 **Settings** 图标。

> 📸 **[截图位置]** 设置页面
> 
> 说明：显示设置项列表

### 通用设置

#### 开机自启

- **启用**: YAT 随系统启动
- **禁用**: 手动启动

**macOS**:
```
系统设置 > 通用 > 登录项
```

**Windows**:
```
任务管理器 > 启动
```

#### 最小化到托盘

- **启用**: 关闭窗口时最小化到系统托盘
- **禁用**: 关闭窗口时退出程序

#### 启动时显示主窗口

- **启用**: 启动时自动显示主窗口
- **禁用**: 启动时在后台运行

## 语言与主题

### 语言设置

YAT 支持多语言：

- ✅ **简体中文** (zh-CN)
- ✅ **English** (en)
- ⚠️ **日本語** (ja) - 开发中
- ⚠️ **Français** (fr) - 开发中

**切换语言**：
1. Settings > 语言
2. 选择语言
3. 界面自动切换

### 主题设置

YAT 支持三种主题模式：

#### Light（亮色）

- 白色背景
- 深色文字
- 适合明亮环境

#### Dark（暗色）

- 深色背景
- 浅色文字
- 适合暗光环境
- 减少眼睛疲劳

#### System（跟随系统）

- 自动跟随系统主题
- 系统切换时自动变化

> 📸 **[截图位置]** 主题切换
> 
> 说明：显示 Light/Dark/System 选项

扩展也会跟随主题切换：

  onThemeChange(isDark, themeMode, context) {
    // 更新扩展 UI
  }

## 数据管理

### 数据位置

YAT 数据存储在以下位置：

**macOS**:
```
~/Library/Application Support/YAT/
├── config.json        # 配置文件
├── tunnels.db         # 隧道数据
├── edges.db           # Edge 数据
└── extensions/        # 扩展目录
```

**Windows**:
```
%APPDATA%/YAT/
```

**Linux**:
```
~/.config/yat/
```

### 清除缓存

如果遇到显示问题，可以尝试清除缓存：

1. Settings > 数据管理
2. 点击 **清除缓存**
3. 重启 YAT

> ⚠️ **注意**：清除缓存不会删除：
> - 隧道配置
> - Edge 配置
> - 账户信息

### 导出数据

导出数据配置（备份）：

1. Settings > 数据管理
2. 点击 **导出配置**
3. 选择保存位置
4. 保存 `.json` 文件

### 导入数据

导入配置（恢复）：

1. Settings > 数据管理
2. 点击 **导入配置**
3. 选择备份文件
4. 确认导入

> ⚠️ **警告**：导入会覆盖当前配置！

## 常见问题

### Q: 忘记密码怎么办？

YAT 使用 OAuth 登录，无需密码。

如果使用邮箱登录：
1. 点击 **忘记密码**
2. 输入邮箱
3. 接收重置链接
4. 设置新密码

### Q: 如何更换绑定的邮箱？

目前不支持更换邮箱，需要：
1. 注销当前账号
2. 使用新邮箱注册
3. 重新配置

### Q: 可以删除账号吗？

可以，但**不可逆**：

1. 联系支持团队
2. 确认删除
3. 所有数据会被清除

> ⚠️ **警告**：
> - 所有隧道会被删除
> - 所有 Edge 会被注销
> - 证书会被撤销

### Q: 多设备登录有限制吗？

**没有限制**。同一账号可以在多个设备登录。

### Q: 数据会同步吗？

**会**。以下数据会跨设备同步：

- ✅ 隧道列表
- ✅ Edge 列表
- ✅ 账户设置
- ❌ 客户端设置（本地）

### Q: 如何保护账户安全？

1. 使用强密码（如果支持）
2. 启用 2FA（即将支持）
3. 定期检查登录历史
4. 不在公共电脑保存登录状态

- 定期检查登录设备
- 及时更新密码
