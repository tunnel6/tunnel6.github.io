## 截图工具推荐

### macOS
- **内置**: Shift + Command + 4（区域截图）
- **推荐**: CleanShot X、Skitch

### Windows
- **内置**: Win + Shift + S（截图工具）
- **推荐**: Snagit、ShareX

### Linux
- **内置**: gnome-screenshot、flameshot
- **推荐**: Flameshot、Shutter
